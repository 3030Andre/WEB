HTML - Linguagem da Web

O que é HTML?

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para estruturar o conteúdo das páginas web. Ele define elementos como títulos, parágrafos, imagens e links.

Papel do HTML na Web

O HTML é responsável pela estrutura da página. Ele organiza o conteúdo, enquanto o CSS cuida da aparência e o JavaScript da interatividade.
